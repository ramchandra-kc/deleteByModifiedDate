# Delete folder recursively by last modified date.

The bat script will check for any files inside said folder that are older than 30 days and if so , they will be deleted.

Please visit here for how to add script to task schedular.  
[schedule task in windows](https://techcommunity.microsoft.com/t5/windows-11/how-to-schedule-any-app-windows-11/m-p/2772955)


I am using this script in a windows task schedular, that will run every week and delete the files and folder older than 30 days for mentioned directory.
This will help you remove unnecessary older files. 

If you want to use this, you can modify the time the folder name and last modification days.
